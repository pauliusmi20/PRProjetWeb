About Fradant :
FRADANT is a dynamic website created by Radja Layaida and Paulius Mickunas, It allows to build different fractals while having fun to change the parameters of construction of these last ones to have them in different positions, different sizes as well as with different colors . FRADANT allows its user to create an account to save the fractals built but also to download them.
